# Show Stash Record

Display full details of a single record.

## Usage

Run `./stash show <id>` to see record details.

## Examples

```bash
# Show record details
./stash show inv-ex4j

# Include attached file contents
./stash show inv-ex4j --with-files

# Show change history
./stash show inv-ex4j --history

# JSON output
./stash show inv-ex4j --json
```

## Instructions

Run `./stash show` with the record ID to display its full details.
